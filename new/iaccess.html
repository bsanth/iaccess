Key Commands

  • Mouse Click - Activate Voice Command
  • Left Button - Go Back to Previous Section
  • Right Button - Advance to Next Section
  • Up Button - Access Table of Contents
  • Down Button - Select Link

Voice Commands

Once you activate voice commands by clicking the left mouse button, you have two main options:

  • To search the web, simply speak the text you wish to search for.
  • To navigate, say go followed by the website you wish to visit.

Go Keywords

The Go voice command has other keywords that provide helpful functions:

  • Go Help takes you to this help page.
  • Go Back takes you to the previous web page.
  • Go Forward takes you to the next web page. This works only if you have previously used the Go Back command.
